# Security Policy

## Supported versions

当前只维护 `main` 分支的最新版本。

## Reporting a vulnerability

请不要为未修复的安全漏洞创建公开 Issue。请通过 GitHub 仓库的 **Security → Report a vulnerability** 私下提交；如果私有漏洞报告尚未启用，请联系仓库维护者 `sxqaq`。

报告建议包含：

- 受影响版本或提交；
- 复现步骤和最小 PoC；
- 可能影响；
- 建议修复方向（如有）。

本项目是朋友房实验平台，尚未经过独立安全审计，不应直接承载敏感账号、支付数据或高价值公共服务。
