## 0.3.0 - 2026-07-17

### 无名杀兼容运行时

- 固定上游真实技能模块可在无 DOM Node 环境加载，并由可快照种子随机源驱动。
- 真实异步技能选择转换为服务端交互请求，支持玩家权限校验、输入日志重放和待交互检查点恢复。
- 高级作者代码增加 Worker 与 VM 双层隔离、执行超时和内存限制。
- `.sgspack` 增加 `noname-compat/v1` 运行时、权限、上游提交和源码哈希清单。
- 高级 Mod 已接入局域网房间权威路径，并由双 WebSocket 客户端端到端测试覆盖发布、进房、开局、命令钩子和一致状态广播。
- 高级 Mod 开局失败现在会回滚房间状态，空对局不会进入自动化或状态广播路径。
- `@sgs/script-sdk` 新增类型安全的 `defineRuntime()`，CLI 新增 `--advanced` 模板并使用服务器同款隔离器执行双次确定性构建测试。
- 高级 `afterCommand` 钩子新增只读命令、核心事件、实际操作者和所选目标上下文，效果目标映射及回放输入哈希同步覆盖这些数据。
- 新增高级 Mod `player-choice` 暂停/恢复协议，支持目标、卡牌、选项、数字、花色选择，以及快照、断线重连、离线 AI、响应校验和不重跑作者代码的回放。
- 153 项工作区测试通过；双 WebSocket 客户端覆盖高级选择请求断线重连与一致结算。
- 固定上游 26 个官方武将技能包全部进入无 DOM 加载回归，白名单处理 `offline/xianding` 的额外导入。
- 插件 CLI 新增 `audit-noname` API 矩阵和 `migrate-noname` 技能迁移骨架命令；报告明确区分加载成功与完整执行兼容。
- Windows 安装包配置携带固定上游的武将规则 JavaScript 切片及 GPL `LICENSE/README`，不打包无关的完整素材库。

## 0.2.1 - 2026-07-17

### 发布工程

- 首个公开安装版统一使用 `0.2.1` 版本号。
- 测试入口改为由 Node.js 枚举 TypeScript 测试文件，可在 Linux、macOS 和 Windows GitHub Actions 中一致运行。

### 自由创作工坊

- 支持一个项目内管理多个武将、技能和卡牌，提供复制、删除、引用联动、撤销、重做与浏览器草稿。
- 支持上传武将立绘和卡面资源，配置模板、焦点、缩放、颜色及技能文字，并导出武将卡 PNG。
- 增加自定义牌、牌堆和模式编辑入口。

### 规则 IR 2.0

- 增加组合条件、状态变量、有界重复、分支、判定、标量选择和连续修正器。
- 增加临时授予/移除技能、跳过阶段、移动牌、失去体力和修改体力上限等效果。
- 所有插件选择、随机行为和效果执行继续进入权威快照、AI 和确定性回放。

### 节点图与开放插件

- 节点图支持稳定节点 ID、入口、显式连线以及循环、断链和不可达节点诊断。
- 服务端记录逐节点执行轨迹，便于复现复杂技能。
- `@sgs/script-sdk` 支持 `rules-ir/v2`；`@sgs/plugin-cli` 使用受限 VM 编译 TypeScript，并拒绝系统导入、动态导入、系统时间和非确定性随机数。

### 局域网扩展分发

- 新增 `.sgspack` 格式，将规则、图片、清单、内容哈希和精确版本依赖封装在一起。
- 房主可下载扩展包，另一台完整安装版可在本地导入并独立开房。
- 支持旧版本并存和依赖保护卸载；被其他扩展依赖的版本不会被误删。

### 验证

- 128 项工作区自动测试通过。
- 10,000 局 2–8 人模拟全部完成，零停滞。
- 双客户端房间、自定义内容和多段插件技能 E2E 通过。
- Windows x64 NSIS 完整安装器构建通过。
