# 朋友联机平台范围

## 已实现

- 无账号游客昵称与稳定设备会话令牌。
- 房间号、邀请链接、密码房、断线恢复、房主转移和闲置房间清理。
- 2–8 人服务端权威身份局、选将、完整固定标准包、AI 托管与确定性回放。
- 节点技能、自定义卡牌、牌堆、模式、胜利条件和高级声明式 SDK。
- 扩展发布前自动测试、不可覆盖版本、分享链接和本地 JSON 导入导出。
- PWA 应用外壳缓存、单端口生产服务、Docker 配置和可持久化内容库。
- Web 平台桥隔离存储、分享和文件保存能力，未来原生外壳无需重写游戏业务。

## 明确不做

- 注册账号、匹配、排位、积分、好友系统和跨设备云存档。
- 反作弊、公开扩展审核、举报、版权工单和公共社区功能。
- 评论、点赞、推荐、作者认证和商业内容运营。
- 当前阶段的 Capacitor/Tauri 安装包。
- 面向公共大服的微服务拆分与横向扩容。

这些裁剪符合“朋友之间在网页上开房游玩”的目标。未来只有在真实出现原生 API 或多实例需求时，才接入 Capacitor、Tauri、共享房间存储和消息总线。
