# 0.3.0 发布说明

0.3.0 把朋友间局域网平台的高级 Mod 从“声明格式”推进到可实际联机使用的服务器权威运行时。

## 主要能力

- 固定并复用 GPL-3.0 无名杀上游源码；26 个官方武将技能包均可在无 DOM Node.js 环境加载。
- 高级 TypeScript `defineRuntime()`、`--advanced` 工程模板、权限与 CPU/内存限制、一次性 Worker + 内层 VM 隔离。
- `roomStart`、`afterCommand`、`choiceResponse` 钩子，提供权威命令、核心事件、实际操作者和目标上下文。
- 目标、卡牌、选项、数字、花色五类玩家选择；支持暂停/恢复、错误响应拒绝、断线重连、离线 AI 和确定性回放。
- 房主通过 `.sgspack` 锁定并分发高级运行时；其他玩家浏览器不执行作者代码。
- `audit-noname` 输出真实 API 兼容矩阵，`migrate-noname` 生成可编译的指定技能迁移骨架。
- Windows x64 安装版内置网页、Host Runtime、326 个上游武将规则 JavaScript 文件和 GPL 许可证；每个安装者都可作为局域网房主。

## 验证

- 153 项工作区自动测试通过。
- 双 WebSocket 客户端验证高级 Mod 发布、入房、开局、玩家选择、断线重连和一致结算。
- 10,000 局 2–8 人身份局模拟零停滞。
- Windows x64 NSIS 安装器本地完整构建通过。
- 本地安装器大小 109,160,894 字节，SHA-256 为 `FE8F758A8F63ACE8C458E12BD4CF1BDA1E499D2F748AA34738D46FA12A7E9D92`；打包程序健康检查退出码为 0。

## 已知边界

- 互联网穿透、中心账号、反作弊和公共 Mod 市场不在本版本目标内。
- 上游技能“可加载”不代表“可完整执行”。静态审计发现 602 种调用形式，其中 556 种仍未映射；目前只有选定真实技能进入逐分支执行验证。
- Worker/VM 不是操作系统容器，只应启用朋友间可信来源的高级代码插件。
- 尚未配置商业代码签名证书，Windows SmartScreen 可能显示未知发布者。
